# Úkoly z ADT
Tento repozitář obsahuje vypracované úlohy ze cvičení a domácí úlohy z předmětu KIV/ADT.

## Struktura repozitáře

### Složka Homeworks
Složka 'Homeworks' obsahuje mnou vypracované domácí cvičení.

### Složka Lessons
Složka 'Lessons' obsahuje podsložky pro každé cvičení a v nich mnou vypracované příklady z daného cvičení.
